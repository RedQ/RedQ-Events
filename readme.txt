Event Management System. Buy sell event tickets online with WooCommerce

== Description ==

Buy or sell online event ticket with the power of Woocommerce Plugin

Major features in RedQ Events include:

* Ticket stock management with WooCommerce stock management system
* Upcoming events display widget
* Add google map location in the event


== Installation ==

To install the plugin go to the wordpress codex: http://codex.wordpress.org/Managing_Plugins or you can install manually http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation

Online documentation: https://github.com/RedQ/RedQ-Events/wiki

== Frequently Asked Questions ==
= FAQ =

Which payment gateway support ?

	- It is build with WooCommmerce so it supports every payment gateway that are supported by WooCommmerce. By default it supports payPal, Direct Bank Tranfer , and MasterCard
